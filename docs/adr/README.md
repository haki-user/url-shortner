# Architecture Decision Records

Create one ADR for each consequential decision that has meaningful alternatives or long-term cost.

Use the next available four-digit number:

```text
0001-short-code-generation.md
0002-url-normalization.md
```

Each ADR should contain:

```markdown
# Decision Title

## Status

Proposed

## Context

What problem and constraints require a decision?

## Decision

What was chosen?

## Alternatives

What credible alternatives were considered?

## Consequences

What becomes easier, harder, or constrained?
```
